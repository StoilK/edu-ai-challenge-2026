# AI Challenge 2.0 - Group 1

Welcome to the AI Challenge 2.0!

We are excited to have you join us and look forward to tackling this challenge together. Below are important notes and instructions to get started.

## 1. GitHub Setup

To submit your tasks, you will need a working GitHub account:

- Make sure you have access to your existing account, or create a new one.
- Create a public repository dedicated to the challenge.
- As you work through the tasks, create a new folder for each task: `task-1`, `task-2`, `task-3`, `task-4`.
- Each task announcement will specify exactly what needs to go inside (solution files, deployed links, artifacts, etc.).

## 4. Scoring and Submission Rules

Each completed and submitted task gives you **100 points**.

If you finish in the top 10, you get bonus points only if the task is fully correct and all requirements are met. If there are mistakes, you are no longer eligible for bonus points for that task, so do not rush.

Last time, a lot of people missed out on prizes because they did not pay attention to details. Org team has no mercy 🙂

Bonus points:

- 1st: `+100`
- 2nd: `+90`
- 3rd: `+80`
- ...
- 10th: `+10`

Double and triple-check the requirements and your solution before submitting. You have exactly one attempt to qualify for bonus points. Any resubmission automatically disqualifies you from bonus points for that task — no exceptions, no appeals, no matter the reason.
