# Task 1 — Leaderboard

Vite + React + TypeScript implementation of the Leaderboard web part (podium, filters, searchable list, expandable “Recent activity” with per-type stats).

## Location

- **App:** `task-1/leaderboard-app/`

## Run

```bash
cd task-1/leaderboard-app
npm install
npm run dev
```

**Production build:** `npm run build` (output in `dist/`; `node_modules` and `dist` are gitignored).
